# mjturt keymap for DZ60

- ISO-layout
- HJKL for arrows

#### Build
```
make dz60:mjturt
```

## Layers:
- Default: (DL, 0)
- Function: Function and navigation keys (FL, 1)
- Control: Backlight and media control (CL, 2)
